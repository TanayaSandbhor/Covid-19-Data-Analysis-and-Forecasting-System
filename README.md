# Covid-19-Data-Analysis-and-Forecasting-System
Policy-aware COVID-19 multi-horizon forecasting pipeline (7/14/28-day) using Wavelet Structural Break Detection, Policy Impact Index, Hybrid ARIMA+RF Ensemble &amp; Peak-Weighted RMSE. Deployed as Streamlit dashboard with 4-level surge alert. Stack: Python · Scikit-learn · Statsmodels · PyWavelets.
